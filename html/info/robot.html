Robot

a.works 레포지토리의 최신 소식을 전합니다.

나의 업무를 도와주는 스마트한 업무 협업자

HOME > 제품소개 > Robot
a.works의 Studio는 사용자가 자동화 프로세스를 설계하는 개발 Tool 입니다. Studio는 프로세스의 복잡성이나 규모에 관계 없이 모든 조직의 비즈니스 프로세스를 모델링하여 자동화 프로세스를 개발할 수 있습니다.

작업 정확성

  • 자동화 프로세스는 설계된 프로세스를 정확하게 수행하는 것이 핵심이며, 이를 위해 객체, 텍스트, 이미지 등에 인식이 정확 하고, 하나 하 나의 단위 업무를 충실히 진행하여 전 체 프로세스를 완료

높은 판단력

  • 자동화된 프로세스를 Robot 진행함에 있어 작업 자의 하드웨어 환경 또는 개인 환경 설정 에 따라 Robot이 스스로 판단하여 작업을 진행 할 수 있는 능력 보유

친숙한 접근성

  • 전사 공통 Task를 Control Room을 통해 공유함으로써 공유된 Task 연결만으 로 업무 Process 구현이 가능
  • Apps Repository에 공유된 Apps를 재활 용하여 개발시간과 비용을 절약 가능
  • a.works Mini를 통해 로컬 PC에서 사용자가 직접 실행, 중지 및 모니터링 가능

주요특징

기능 강약 조절
항상 노출되어야 하는 기능을 모아 간결하고 심플한 형태로 제공하고, 맥락에 따라 필요한 기능들은 필요 시 확장하여 이용할 수 있도록 Drawer 형태로 제공
액션 가이드
프로그래밍 관련 전문 지식이 없는 일반사용자에게 진입 시 어떤 액션을 해야 하는지에 대한 가이드를 제공
넓은 캔버스 영역
주요 기능 위주로 스튜디오를 간결하게 구성하여 넓은 캔버스 영역을 확보하 고, 심플한 이미지를 통해 기존 RPA 프로그램 보다 쉽고 친근한 첫인상을 제공
Drag&Drop + a
일반 사용자를 위한 Drag&Drop 방식의 직관적인 UI가 숙련도가 높은 전문 개발자의 생산성을 해치지 않도록 필요 시 텍스트 입력을 통해 이벤트를 빠르 게 추가할 수 있게 제공
자동 완성 기능
숙련도가 낮은 일반 사용자에게는 잘못된 입력을 예방하고, 동시에 입력값을 쉽게 선택할 수 있도록 돕는 자동 완성 기능을 제공
Hybrid 구조
순차 - 분기가 반복되는 사용자의 업무 흐름을 반영해 Flowchart와 List구조를 혼합한 Hybrid 형태로 구성하여 구조와 흐름 파악이 용이하도록 설계
핵심 Property 탐색
이벤트가 가지고 있는 다수의 Property 중, 개발 시 자주 확인 하는 Property 항목을 표기하여 스크립트를 전체 시야에서 확인 할 때도 빠르게 속성을 파악 하고, 스크립트 탐색 효율을 높일 수 있도록 설계.
미니맵
현재 화면의 위치를 전체적인 시야에서 파악하고 원하는 위치에 바르게 접근 할 수 있도록 미니맵을 제공
통합 탐색
필요 시 화면을 확장하여 확인 및 접근이 필요한 주요 요소들 (Validate, Variable, Group, Break Point 등)을 한 곳에 모아서 빠르게 확인하고 접근할 수 있도록 배치
Validate 체크
숙련도가 낮은 사용자가 실행 전에 체크할 수 있는 오류 사항을 미리 확인하 고, 대처 가이드를 함께 제공받아 손쉽게 수정에 반영할 수 있도록 Validate 체크 기능을 제공
Screenshot drawer
Front-end 중심으로 흐름이 짜여지게 되는 RPA 프로그래밍의 특성을 고려하 여, 화면 흐름 위주로 모아서 탐색할 수 있는 기능 제공해 개발 효율을 높였음
네비게이션
다른 사람이 작성한 스크립트를 열었을 때, 빠르게 전체 구조를 파악하고, 원하는 수정 위치와 방법을 파악할 수 있도록 Group 표현 방식 및 Right drawer를 활용한 내비게이션 기능을 제공
Repository
일반 사용자가 Repository 기능과 컨셉을 쉽게 이해하고, 프로그래밍에 대한 거리감을 줄일 수 있도록 Template 이란 용어를 사용하고, 익숙한 엑셀이나 파 워포인트와 유사하게 느낄 수 있도록 화면을 구성. 또한 Repository Task/Process를 이벤트가 제공되는 Palette 영역에 탭 전환 UI으로 제공하여 이벤트를 추가하듯 직관적으로 스크립트 재사용 작업을 진행할 수 있도록 구성
녹화 진행상태 확인
사용자 행동 Recording을 통한 스크립트 작성 시, 액션이 스크립트에 반영 되 었는지에 대한 불안 없이 진행할 수 있도록 액션 반영 상태에 대한 명확한 피 드백을 제공
디버깅
디버깅시 단계별 실행, 부분 실행, 전체 실행 후 오류 정보 확인이 가능하고, 디버깅 실행 시 실시간으로 변수값의 변경 내용을 확인 가능. 또한 잘못된 변 수값이 발생하는 지점을 즉시 Tracking 가능할 뿐만 아니라, Break point를 지 정하여 중단 지점을 설정하거나 해제 가능
사용자 정의 코드 지원
RPA 개발자는 사용자 정의 C# (V1.0에서 지원) 및 Python/Java/Powershell V2.0에서 지원)을 자동화 스크립트에 직접 통합하여 보다 복잡하고 전문적인 프로세스 자동화를 구현 가능. 이 결과물은 Repository 에 저장하여 전사에 공 유하여 재사용 가능.
